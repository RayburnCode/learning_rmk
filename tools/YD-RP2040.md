#

The YD-RP2040 is an improved, budget-friendly clone of the Raspberry Pi Pico, featuring the RP2040 microcontroller (dual-core ARM M0+, 133MHz, 264KB SRAM). It offers upgrades like a USB-C connector, 4MB to 16MB flash memory, a reset button, user button (GPIO24), and an onboard WS2812 RGB LED (GPIO23)
